# Dataset: vehicle_positions

## Informações gerais
- Layer: Silver
- Source: SPTrans API
- Owner: Data Engineering
- Update Frequency: Near Real-Time
- Data Classification: Public

## Descrição
Dataset com posições de veículos em operação, processado a partir da API da SPTrans.

## Campos
### event_date
Data do evento.

### event_hour
Hora do evento.

### line_id
Identificador da linha.

### vehicle_id
Identificador do veículo.

### latitude
Latitude do veículo.

### longitude
Longitude do veículo.

### lat_grid
Grid geoespacial de latitude.

### lon_grid
Grid geoespacial de longitude.

### accessible
Indica se o veículo possui acessibilidade.

## Observações
Esse dataset alimenta KPIs, heatmaps e análises de mobilidade urbana.
