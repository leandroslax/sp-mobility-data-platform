# Governance

Esta pasta centraliza os artefatos de governança da plataforma de dados de mobilidade urbana.

## Estrutura

- data_dictionary: dicionário de dados dos principais datasets
- policies: políticas de acesso, retenção, classificação e ownership
- lineage: documentação de linhagem dos dados
- quality: regras de qualidade de dados

## Objetivo
Apoiar boas práticas de engenharia de dados, governança, rastreabilidade e padronização da plataforma.
